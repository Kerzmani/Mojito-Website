## Mojito Website Clone

Modern design website that displays proficiency in HTML, React, and JS. For this project, I also learned how to use GSAP for beautiful, smooth animations and Tailwind CSS for 
excellent responsive design, cleaner code, and quicker UI development. 

## Features

- Responsive design
- Smooth GSAP animations
- Modern UI built with Tailwind CSS
- Fast development environment using Vite

# Tech Stack

- React
- Vite
- Tailwind CSS
- GSAP
- HTML5
- JavaScript (ES6+)
